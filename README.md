# 💡 Lumière Control

Sistema web de gestão para clínicas de estética, desenvolvido como projeto acadêmico e aplicado como solução real pela FluxON.

---

# 🚀 Sobre o Projeto

O **Lumière Control** é um sistema web criado para organizar e estruturar a operação da **Lumière Clinic**, uma clínica de estética que enfrenta desafios no controle de clientes, agendamentos e serviços.

Este projeto faz parte da proposta da **FluxON**, uma iniciativa focada em transformar negócios desorganizados em operações eficientes, estruturadas e preparadas para crescimento.

---

# 🎯 Problema

A Lumière Clinic possui dificuldades na organização operacional, como:

* Controle de clientes descentralizado
* Agendamentos feitos manualmente
* Falta de histórico de atendimentos
* Dificuldade no controle de serviços

Esses problemas geram desorganização, perda de tempo e dificultam o crescimento do negócio.

---

# ✅ Solução

O Lumière Control propõe um sistema web que:

* Centraliza o cadastro de clientes
* Organiza agendamentos
* Controla serviços realizados
* Mantém histórico de atendimentos
* Oferece uma visão geral da operação

---

# 🧩 Funcionalidades (MVP)

* Cadastro de clientes
* Cadastro de serviços
* Agendamento de atendimentos
* Visualização de agenda
* Histórico de atendimentos
* Dashboard simples

# ⚙️ Como Executar o Projeto

## Pré-requisitos

* Java 21+
* Node.js
* MySQL
* Git

---

## 🔧 Backend

```bash
cd backend
./mvnw spring-boot:run
```

---

## 💻 Frontend

```bash
cd frontend
npm install
npm run dev
```
